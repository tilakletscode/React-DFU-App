# DFU Mobile Application

## Overview

This project is a mobile application for Diabetic Foot Ulcer (DFU) prediction.  

---

## Technology Stack

**Frontend**
- React Native (Expo)

**Backend (Architecture Prepared)**
- Node.js (API Layer)
- Python (Flask – ML Inference Server)

**Database**
- Cloud-based database

**Authentication**
- JWT-based role structure
- User roles: Patient / Doctor / Admin

---

## Application Modules (In Development)

- User Registration & Login Interface
- Role-Based Access Structure
- Patient Dashboard UI
- Doctor Dashboard UI
- Image Upload Interface (UI Layer)
- ML Integration Pipeline (architecture prepared)
